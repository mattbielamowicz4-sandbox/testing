# Simple Timer App

Goal:
Create a simple web timer app that:
- Lets the user start, stop, and reset a countdown timer.
- Shows minutes and seconds.
- Works in the browser (no backend needed).

Done Criteria:
- A homepage (index.html).
- A style sheet (styles.css).
- A JavaScript file (script.js).
- Buttons for start/stop/reset.
- Timer display updates every second.

Non-goals:
- Do not add build tools (like webpack).
- Keep it simple plain HTML/CSS/JS.
